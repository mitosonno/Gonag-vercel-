# GONAG.AZ

Azərbaycan toy və məclis idarəetmə sistemi.

## Deploy addımları

1. Bu qovluğu GitHub-a yüklə
2. vercel.com-da "New Project" → GitHub repo seç
3. Avtomatik deploy olur

## Güncəlləmə

1. Claude-dan yeni `gonag.jsx` endir
2. `src/App.jsx` faylını yeni versiya ilə əvəz et
3. GitHub-a push et → Vercel avtomatik yenilənir
